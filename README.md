cion-clojure
============

cion will be a tool for configuring ci servers.
At the moment, it will be written in clojure and support TeamCity.

## Installation

    $ git clone https://github.com/ivanmoore/cion-clojure
	
Also - install http://leiningen.org/

## Usage

    $ lein run
